A robust, full-stack Workforce Management & Shift Tracking Platform.

WorkSync is a unified dashboard designed for agencies and startups to streamline intern management and staff productivity. Built with performance and security in mind, it bridges the gap between task assignment and real-time accountability.

🚀 Key Features:

Role-Based Workflows: Separate, secure dashboards for Administrators and Staff/Interns.
Real-Time Shift Management: One-click clock-in/out system with shift notes for audit trails.
Centralized Task Allocation: Admins can assign, track, and monitor tasks assigned to specific users.
Data Persistence: Integrated with Supabase (PostgreSQL) for reliable and scalable data handling.
Secure Auth: Cookie-based authentication middleware ensuring high-level security for internal operations.
🛠 Tech Stack:

Frontend & API: Next.js (App Router), React, Tailwind CSS
Database & Auth: Supabase
Icons & UI: Lucide React
